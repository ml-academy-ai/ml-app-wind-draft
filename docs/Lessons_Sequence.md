## Lesson Sequence

1. **Lesson 0: Directory Setup and UV Installation** - Create project directory, install uv, and set up virtual environment
2. **Lesson 1: EDA Setup** - Set up environment for exploratory data analysis with Jupyter notebooks
3. **Lesson 2: Code Quality** - Pre-commit hooks, linting, and code formatting
4. **Lesson 3: Data Management** - Data versioning and management strategies
5. **Lesson 4: Kedro Project Setup** - Set up Kedro project structure (after EDA is complete)
6. **Lesson 5: Kedro Pipelines** - Building and organizing data pipelines
7. **Lesson 6: Dash UI** - Creating interactive dashboards
8. **Lesson 7: Pytest Tutorial** - Writing and running tests
9. **Lesson 8: Docker Containerization** - Containerizing the application
10. **Lesson 9: CI GitHub Actions** - Setting up continuous integration
11. **Lesson 10: Continuous Deployment** - Automated deployment workflows
12. **Lesson 11: Prefect** - Workflow orchestration with Prefect
